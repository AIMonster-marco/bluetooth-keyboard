# Bluetooth Keyboard

안드로이드 블루투스 키보드 앱입니다. 이 앱을 사용하면 안드로이드 기기를 블루투스 키보드로 사용할 수 있습니다.

## 기능

- 블루투스 장치 검색 및 연결
- 실시간 텍스트 입력 및 전송
- 블루투스 상태 모니터링

## 사용 방법

1. 앱을 실행합니다.
2. "블루투스 연결" 버튼을 눌러 블루투스 장치를 검색합니다.
3. 연결할 장치를 선택합니다.
4. 텍스트 입력 영역에 텍스트를 입력하면 자동으로 연결된 장치로 전송됩니다.
